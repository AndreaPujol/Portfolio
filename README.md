# Portfolio Andrea Pujol - Diseñadora UX/UI

## 🎨 Descripción del Proyecto

Portfolio profesional de Andrea Pujol, diseñadora UX/UI especializada en experiencias digitales, identidad visual, diseño editorial y packaging. Este sitio web moderno y responsivo presenta sus proyectos más destacados con un enfoque en la usabilidad y la estética visual.

## ✨ Características Principales

### 🎯 Diseño y UX
- **Diseño responsive** optimizado para todos los dispositivos
- **Interfaz moderna** con animaciones suaves y transiciones elegantes
- **Navegación intuitiva** con scroll suave y menú sticky
- **Accesibilidad** mejorada con contraste adecuado y navegación por teclado

### 🎨 Funcionalidades Destacadas
- **Portfolio interactivo** con filtros por categoría
- **Modales detallados** para cada proyecto con información completa
- **Visualización de datos** con gráficos interactivos de paletas de colores
- **Animaciones on-scroll** para una experiencia envolvente
- **Menú móvil** funcional para dispositivos pequeños

### 🛠️ Tecnologías Utilizadas
- **HTML5** semántico y estructurado
- **CSS3** con variables personalizadas y Flexbox/Grid
- **JavaScript ES6+** modular y optimizado
- **Tailwind CSS** para estilos utilitarios
- **Chart.js** para visualización de datos
- **Google Fonts** (Inter + Playfair Display)

## 📝 Notas de Desarrollo

### Optimizaciones Implementadas
- **Lazy loading** para imágenes de proyectos
- **Preloading** de assets críticos
- **Minificación** de animaciones CSS
- **Debouncing** en eventos de scroll
- **Caching** de instancias de Chart.js

### Accesibilidad
- **Navegación por teclado** completa
- **Screen readers** compatibles
- **Contraste** WCAG AA compliant
- **Focus indicators** visibles
- **Alt text** descriptivo en imágenes

### Performance
- **CSS Grid y Flexbox** para layouts eficientes
- **Transform3d** para animaciones de hardware
- **Event delegation** para mejor rendimiento
- **CSS variables** para theming consistente
